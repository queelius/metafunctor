Project Description

The R package femtograd provides a way of doing automatic differentation. It’s not particularly fast, as it was just an experiment, but I may end up using it in my likelihood.model or algebraic.mle packages.

I will probably optimize it and provide a C++ implementation using Rcpp, but I don’t want to spend too much time on it since there are already nice packages that do this.
